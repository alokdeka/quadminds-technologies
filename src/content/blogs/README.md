# Blog Content

Drop your `.md` blog post files in this folder.

Each file should start with a YAML frontmatter block:

```markdown
---
title: "Your Post Title"
slug: "your-post-slug"
date: "2026-06-02"
author: "Alok Deka"
excerpt: "A short summary shown in blog cards."
tags: ["tag1", "tag2"]
cover: "/blog/your-cover-image.jpg"
---

Your markdown content starts here...
```

Cover images go in `public/blog/`.
